Air Quality Monitoring System

This project uses Mobile Sensor Cloud Computing to implement the Air Quality Monitoring System.

  • Designed and developed mobile cloud components to establish mobile sensor cloud infrastructure with simulated sensors and sensor controllers by using AWS, Node.js, Android, MongoDB and MySQL.
  • Implemented Load Balancer to provide scalability to the model and used CloudWatch API to monitor AWS instances.
  • Screen air quality by accepting the sensor (which be static or portable) information from a cloud.
  • Sensor is joined to the mobile utilizing bluetooth. Mobile will get concentration of pollutants from the sensor.
  • Sensor's data is sent to controller which uses Round Robin Algorithm to push the sensor data into database.
  • User module consists of billing module and sensor data information management. User can check his/her billing information as per data used and also got the functionality of data management related to sensors. A user can add, delete and update sensors from his account as per the choice of plan. He can view sensor data in graph format and even download the same data in the image or pdf format.
  • A user can see the data from sensors in graphical format through a single click on data button.
